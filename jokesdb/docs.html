Instalation

You can install JokesDB via pip:

pip install JokesDB

Example of Use

Basic usage in Python:


    from jokesdb import get_jokes, joke_of_the_day

    # Get one random joke
    print(get_jokes())

    # Get 5 jokes from a specific category
    print(get_jokes(number=5, category="fun"))

    # Get the joke of the day
    print(joke_of_the_day())
